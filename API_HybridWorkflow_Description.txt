1. Purpose
-----------
This is a demo skeleton of the API. It allows your app to submit data and store it in a local folder.

2. Usage
--------
- POST /data/submit
- Parameters:
    - app_name: name of your app
    - payload: data to store
- Returns:
    - saved_path: path of saved file
    - is_sensitive: always False in demo
    - next_actions: ["auto_save"]
